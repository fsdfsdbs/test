Gun.Lol Clone

Welcome!

This is a complete, open-source profile website inspired by gun.lol and gunslol-open-source.

This test page helps you verify the setup before adding real assets.

Features

Multiple Themes

5 different themes with custom backgrounds and music

Animated UI

Smooth animations, typing effects, and transitions

Custom Cursor

Beautiful animated cursor with glow effects

Skills Display

Animated skill bars with percentages

Projects Showcase

Display your projects with cards

Settings Panel

Customize username, bio, and colors

Setup Instructions

  1. Add Assets: Place your files in the assets/ folder
  2. Required Files: See README.md for the complete list
  3. Customize: Edit config.json or script.js to change settings
  4. Run: Open index.html with a live server

Note: For a live server, use VS Code Live Server extension or run: python -m http.server 8000

Keyboard Shortcuts

M

Files Included

  • index.html - Main page
  • style.css - All styles
  • script.js - All JavaScript
  • config.json - Configuration
  • README.md - Documentation
  • assets/ - Folder for your media files
  • test.html - This test page
  • generate-assets.sh - Generate placeholder assets
